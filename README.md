# Platformer AI Game

A simple web platformer with:

- 3 playable levels
- A Play-only main menu
- Flag-to-next-level progression
- Basic jump/hit/flag audio plus background melody

## Run locally

Open `index.html` in a browser.

For best results with local file restrictions, run a small static server:

```powershell
cd platformer-ai-game
python -m http.server 8080
```

Then open http://localhost:8080

## Controls

- Move: A/D or Left/Right
- Jump: W, Up Arrow, or Space
